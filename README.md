# WhatsApp Replica
The WhatsApp popular messenger web version replica

```
Used Stack:
Frontend: JS (ES6), Vue.js, Socket.io, SCSS
Backend: Node.js, Socket.io
DB: MongoDB
```

### App features
- Chat one-to-one
- Adding contacts from all existing users
